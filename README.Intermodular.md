**Información acerca del proyecto intermodular.**



Tienda de Videojuegos - Bit-Love

##### **Descripción**

Bit-Love es un proyecto intermodular de 1º DAW que consiste en el desarrollo de una tienda online de videojuegos en formato físico.



El objetivo del proyecto es simular el funcionamiento de una tienda real, permitiendo la visualización de un catálogo de productos, la gestión de pedidos y el almacenamiento de información mediante una base de datos.



La aplicación integra una parte visual (web), una aplicación desarrollada en Java y una base de datos relacional conectada mediante JDBC, permitiendo la gestión de la información de la tienda y simulando un entorno real de desarrollo.







##### **Objetivo del proyecto**



El objetivo del proyecto es aplicar de forma integrada los conocimientos adquiridos en los distintos módulos del ciclo de Desarrollo de Aplicaciones Web, simulando un entorno profesional real.



A través del desarrollo de la tienda online Bit-Love, se trabajan los siguientes módulos:



* ###### **Lenguajes de Marcas y Sistemas de Gestión de Información:**

Desarrollo de la interfaz web mediante HTML y CSS, creando una estructura semántica, navegación funcional y un diseño visual coherente y responsive.





* ###### **Bases de Datos**

Diseño y creación de una base de datos relacional que modela el funcionamiento de la tienda, incluyendo entidades como productos, usuarios y pedidos, así como sus relaciones.





* ###### **Programación**

Desarrollo de una aplicación en Java que permite gestionar la información de la tienda (productos, pedidos, etc.), implementando lógica de negocio y operaciones CRUD.





* ###### **Entornos de Desarrollo**

Uso de herramientas de control de versiones como Git y GitHub, organización del proyecto en repositorio y seguimiento del desarrollo mediante commits estructurados.





* ###### **Sistemas Informáticos**

Definición del entorno de ejecución necesario para el proyecto, incluyendo sistema operativo, requisitos de hardware, instalación de software y configuración básica.





* ###### **Módulo Profesional Optativo (MPO) – Ampliación de Programación**

Mejora de la estructura del código aplicando principios de programación orientada a objetos, organización en capas y buenas prácticas de desarrollo.





* ###### **Itinerario Personal para la Empleabilidad I**

Desarrollo de un perfil profesional inicial, presentación del proyecto como parte de un portfolio y reflexión sobre los conocimientos adquiridos y las competencias desarrolladas.
