### Informe de Sistemas – Bit-Love





##### Entorno de ejecución



La aplicación Bit-Love se ejecuta en un entorno local, en un equipo personal.



Este entorno permite trabajar con la aplicación Java y la base de datos en el mismo equipo, facilitando el desarrollo y las pruebas del proyecto.





##### Tipo de sistema

* PC de usuario

##### 

##### Sistema operativo

* Windows 10 o superior





##### Software necesario

* Java JDK
* XAMPP (Apache + MySQL)
* phpMyAdmin (gestión de la base de datos)
* IDE de desarrollo (IntelliJ IDEA)





###### **Base de datos**

La base de datos se utiliza mediante MySQL local, incluido en XAMPP, y se gestiona con phpMyAdmin, lo que permite crear tablas y gestionar los datos de forma sencilla.

La base se accede desde la aplicación Java mediante JDBC.
